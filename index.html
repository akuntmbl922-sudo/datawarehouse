API Documentation: Dimensi Data

Selamat datang! API ini menyediakan data dimensi dalam format JSON.

🔗 Cara Akses

Gunakan metode GET pada URL berikut: [URL_SERVER]/[NAMA_TABEL.json]

📋 Daftar Endpoint

EndpointKeterangan
/dim_jabatan.jsonData master jabatan
/dim_kabupaten_kota.jsonData master wilayah
/dim_lama_dapat_kerja.jsonData durasi kerja
/dim_sekolah.jsonData master sekolah
/dim_prodi.jsonData master program studi

[
    {
        "ID_JABATAN": 1,
